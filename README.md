# Task 03: Decision Tree Classifier

## 📌 Objective
Build a Decision Tree model to predict whether a customer will purchase a product/service.

## 📂 Dataset
Sample dataset:  
https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%203

## 🛠️ Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🔍 Steps Performed
1. Data preprocessing:
   - Encoding categorical variables
   - Handling missing values
2. Split dataset into training & testing sets
3. Trained Decision Tree Classifier
4. Evaluated model:
   - Accuracy score
   - Confusion matrix
5. Visualized decision tree

## 📊 Output
- Prediction of customer purchase behavior
- Model accuracy evaluation

## 📌 Conclusion
Decision Trees help in understanding decision-making logic and provide interpretable results.
